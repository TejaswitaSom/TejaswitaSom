1. git diff
2. git add -p  --> patch addition, ie parts of code, for adding specific hunk, press s after that
3. git log --oneline
4. git commit --amend 
5. git restore --staged .
6. git rebase -i HEAD~2 --> interactive rebase, --> further can be used to squash commits in vim 
7. git rebase --abort    --> any abort command of this type
8. git reflog --> inspect the record of events at a granular level
9. git reset --hard HEAD@{6}  -> to rollback rebase upto a particular commit , here HEAD@{6} generated by reflog, which was lost during rebase
